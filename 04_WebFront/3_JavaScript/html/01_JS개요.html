JS 개요

스크립트(script) 언어

    ex) 자바스크립트, 파이썬
    기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도의 언어
    
    별도로 소스코드를 컴파일하지 않고
    인터프리터(interpreter, 한 줄씩 코드를 읽어서 해석)
    를 이용해서 코드를 바로 실행
    -> 장점 : 컴파일에 소요되는 시간이 없음
    -> 단점 : 프로그램 실행 시간이 상대적으로 느림

    짧은 소스코드 파일로 상호작용 하도록 고안됨

자바스크립트(JavaScript)

    웹 브라우저에서 주로 사용하는
    인터프리터 방식의 객체 지향 프로그래밍 언어이다.

    (이름에 Java 들어갔다고 해서 Java랑 관련있는 것 아닙니다...)

    ECMA Script 따르는 대표적인 웹 기술
    (국제적 표준 기구의 기준에 맞게 만들어짐)
    (ES5, ES6, ...)

JavaScript 작성 방법

  • 브라우저 개발자도구 -> 콘솔(console)에 직접 작성하기
  • html 내부에 script 태그를 이용해 작성(internal 방식)
  • html 외부 .js 파일에 작성 후 연결해서 사용(external 방식)
  • 태그에 직접 JS코드를 작성(inline 방식, 권장 X 수정하기힘들어서 테스트할때나 사용)
