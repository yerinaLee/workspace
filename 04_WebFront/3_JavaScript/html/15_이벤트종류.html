JS 이벤트 종류

focus : 초점이 맞춰졌을 때

focus-out(추천x!)

blur : 초점을 잃었을 때

change : input의 값이 변했을 때

    - checkbox, radio : 체크 됐을 때 / 해제 됐을 때
                        change 이벤트 발생

    -text, number 관련 :
     input의 초점을 잃었을 때(blur)
     이전 값과 현재 값이 다를 경우 change 발생
     (값의 변화를 인지함 (실시간 x))

input 이벤트

    - input 요소에 값이 입력되는 모든 동작을 인식해 발생하는 이벤트
    
    - 단점 : 키보드 관련 이벤트는 입력한 키를 알 수 있지만
            (e.key로 얻어와짐)
            input 이벤트는 e.key 사용 불가
            (이벤트 객체 e에 key 속성이 존재하지 않음)
