기본 속성 선택자

    태그에 작성된 특정 속성을 선택하는 선택자
    (모든 속성을 선택할 수 있는 기본 방법)
    (id, class도 선택 가능하지만 보통은 #, . 을 사용)

    [작성법]
    선택자[속성명="속성값"] { CSS코드; }

    (참고)
    1) 선택자는 생략할 수 있다.
       -> 특정 속성을 가진 모든 요소를 선택

    2) "속성값"에서 쌍따옴표("")는
      홑따옴표('')로 변경하거나 생략할 수 있다

자식 선택자( > )

    지정된 요소 바로 한 단계 하위 요소를 선택하는 선택자

    [작성법]
    선택자1 > 선택자2 { CSS 코드; }

    - 선택자1 : 부모 요소
    - 선택자2 : 자식 요소(한 단계 하위 요소)
  • 자식1
  • 자식2
  • 자식3
  • 자식4

후손(자손) 선택자 (작성법: 띄어쓰기)

    지정된 요소의 모든 하위 요소를 선택하는 선택자

    (자식 : 1단계 하위)
    (후손 : n단계 모든 하위)

    [작성법]
    선택자1 선택자2 { CSS 코드; }

    선택자1 : 부모(조상) 요소 선택자
    선택자2 : 후손 요소 선택자

반응 선택자

    사용자의 움직임에 반응하여 스타일이 변경되는 선택자

    선택자:active -> 선택된 요소가 클릭되어지고 있을 경우

    선택자:hover  -> 선택된 요소에 마우스가 올라가 있는 경우
:active 테스트
:hover 테스트

상태 선택자

    입력 양식(input, input 관련 태그)의
    상태에 따라 선택되는 선택자

    :focus -> 요소에 초점이 맞춰졌을 때 (커서 깜빡일때)
    :checked -> 요소가 체크되었을 때(radio, checkbox)
    :enabled / :disabled -> 요소가 활성(입력가능상태)/비활성 상태(입력불가상태)일 때

동위 선택자

    동위(동등한 위치 == 형제 관계)관계에서 
    다음(뒤)에 위치한 요소를 선택하는 선택자 (다음만 찾을 수 있음! 앞은 JS만 가능)

    A
    B
    B

    1) A바로 다음(뒤)에 있는 B 요소를 하나 선택(+)
    A선택자 + B선택자 { CSS 코드; }

    2) A뒤에 있는 모든 B요소를 선택(~ 틸드(Tilde))
    A선택자 ~ B선택자 { CSS 코드; }
