요소 정렬 스타일(float / clear)

    float(뜨다, 띄우다, 뜨는 물건) : 요소를 띄워서 좌/우로 정렬하는 속성
    
    clear : float로 인해 띄워져 있는 상태를 해제하는 속성

    (float 사용 시 요소가 겹치는 문제를 clear로 해결)
