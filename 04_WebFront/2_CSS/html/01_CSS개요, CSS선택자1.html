CSS(Cascading Style Sheets) 

    마크업 언어(HTML)가 실제로 화면에 표시되는 방법을 기술하는 언어
    (모양, 색상, 크기, 스타일 ... 지정 가능)

    하나의 요소에 여러 CSS코드를 겹쳐서 원하는 스타일을 만들 수 있다

    W3C에서 CSS를 웹 표준으로 지정함

CSS 선택자(Selector)

    현재 HTML 문서 내에서 특정 요소를 선택하는 문자 또는 문자열(기호)

    (스타일을 적용하고자 하는 요소를 선택하는 방법)

    요소 : 시작태그 + 속성 + 내용 + 종료태그
    
    ** CSS 코드를 작성하는 방법 **
    1. head 태그에 style 태그를 이용해서 작성하는 방법
    2. 외부에 css 파일을 별도로 생성하여 연결하는 방법

태그 선택자

    현재 HTML문서 내에 같은 태그를 모두 선택하는 선택자

    [작성법]
    태그명 { CSS코드; }
strong 태그에 작성된 내용입니다.
span 태그에 작성된 내용입니다
  • strong 태그에 작성된 내용입니다.
  • span 태그에 작성된 내용입니다

id 선택자 (#id속성값)

    HTML문서 내에서 id 속성값이 일치하는 요소를 선택하는 선택자

    (중요!!)
    id 속성 값은 페이지 내에서 유일해야 한다!!
    (제발 중복해서 쓰지마요 ㅠㅠ)

    [작성법]
    #id속성값 { css 코드; }

클래스 선택자(.class속성값)

    HTML문서 내에서 클래스 속성값이 일치하는 요소를 선택하는 선택자

    [작성법]
    .class속성값 { CSS 코드; }

    ** 요소에 CSS를 적용할 때에는 id보단 class를 이용하는 것이 좋다 **
    (css는 대부분 class사용. id는 JS에서 주로 사용)
